BlueWhaleYT2024 年 01 月 09 日2024 年 01 月 10 日

選擇器

CSS 可以使用以下選擇器進行鎖定目標的樣式設定：

  • 標記
  • 類別 .
  • ID #
  • 複合選擇器

複合選擇器

當選擇條件不足，就要符合複合條件指定。

子孫選擇

  • 後裔選擇器 (Decendent Selector) X Y : 用於選擇某個元素的後代元素。後代元素可以是直接子元素，也可以是更深層次的子孫元素。
  • 子選擇器 (Child Selector) X > Y : 用於選擇某個元素的直接子元素，而不包括更深層次的子孫元素。

相鄰選擇 X + Y

相鄰兄弟（單選）僅僅會選擇剛好在 X 元素之後的 Y 元素。

相鄰群選擇 X ~ Y

相鄰兄弟群（多選）僅僅會選擇剛好在 X 元素之後的同層 Y 元素。

擬態類別

擬態類別，又稱偽類別，部分的標籤會提供其他特殊情況下的條件，稱呼為擬態類別並使用冒號 : 表示。

參考資料

假設以 3 作為索引：

選擇器 說明
p:nth-child(3) 找到 p 是某父級之所有 child 範圍內，第 3 個 (索引最小為 1)
p:nth-child(3n+1) 找到 p 是某父級之所有 child 範圍內，第 1, 4, 7 … 個
p:nth-child(3n) 找到 p 是某父級之所有 child 範圍內，第 3, 6, 9… 個
p:nth-child(-n+3) 找到 p 是某父級之所有 child 範圍內，第 1, 2, 3 個
p:nth-child(odd) 找到 p 是某父級之所有 child 範圍內，第單數個 (1,3, 5 …) 等價 2n+1
p:nth-of-type(3) 找到這些 p 的父級之第 3 個兒子 p 。(索引最小為 1)
p:nth-of-type(3n+1) 找到這些 p 的父級之第 1, 4 ,7 … 兒子 p 。
p:nth-of-type(3n) 找到這些 p 的父級之第 3, 6, 9 … 兒子 p 。
p:nth-of-type(odd) 找到這些 p 的父級之偶數兒子 p 。
p:nth-last-of-type(3) 找到這些 p 的父級之倒數第 3 個兒子 p 。(索引最小為 1)
p:first-child 找到 p，且需符合為某父級下之第 1 個兒子。
p:last-child 找到 p，且需符合為某父級之最後個兒子。
p:nth-last-child(3) 找到 p，且需符合為某父級之倒數第 3 個兒子 p。(索引最小為 1)

擬態元素

跟擬態類別語法差不多，不過要用雙冒號 :: 表示。

雖然，只以一個冒號來定義擬態元素，瀏覽器寫也可以正常判斷運作 (唯獨 ::selection 必須是兩個冒號才能正常運作)，不過為了區分擬態類別及擬態元素，用兩個冒號為宜。

前置及後置偽元素

要為目標元素定義前置內容或後置內容，可以使用 ::before 定義前置，::after 定義後置。

content 屬性是 ::before 與 ::after 必須具備的屬性，如果不需要用到文本內容，可填 "" 表示。

content 屬性可以配合使用 attr() 方法獲取內容元素的屬性值。

其他選擇器

p::selection 樣式會被應用於文字被選取的時候
p::first-letter 只應用樣式於第一個字母
p::first-line 只應用樣式於第一行文字
