因為 Express 就是以路由構建的形式編寫的，所以當要實作一個 RESTful API，使用 Express 是較 Node.js 為簡單的。

API HTTP 方法 作用
listUsers GET 獲取所有用戶
getUser GET 獲取指定用戶
addUser POST 新增用戶
deleteUser DELETE 刪除指定用戶

本地測試或部署後，可以使用 Postman 或其他工具來測試 API。這裏推薦使用 Postman 軟件。
