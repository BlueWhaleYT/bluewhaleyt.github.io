Starting from Android 11, Android/data & Android/obb files are not able to be accessed through general external storage uses. However, the Android Official doesn’t block out the way to access those files, that is the way we could wield is Storage Access Framework (SAF). Therefore, we will learn how to list all files of Android/data using SAF in this article.

With SAF, we don’t need any permissions from AndroidManifest.xml to access the files of external storages.
