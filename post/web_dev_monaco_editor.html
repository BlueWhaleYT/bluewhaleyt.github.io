什麼是 Monaco Editor？

Monaco Editor 是一個代碼編輯器的函式庫組件，提供一個強大的編輯器功能，這個編輯器亦得到 Visual Studio Code (簡稱：VSCode) 的「賞識」，現在 VSCode 都是在用這個編輯器。

但是，Monaco Editor 目前不支援響應式設計，如果你試圖在手機裝置上使用它，可能在輸入方面，或是整體的使用體驗不佳。
